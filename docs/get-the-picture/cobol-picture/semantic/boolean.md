# 布林

⚠️ `USAGE` 僅支援 `DISPLAY`。

## Decode（DISPLAY → bool）

支援類型  

| PIC 類型 | 輸入內容       | 結果       | 備註        |
| -------- | ------------- | ---------- | ----------- |
| `9(1)`   | `"1"`         | `true`     | ASCII 0x31  |
| `9(1)`   | `"0"`         | `false`    | ASCII 0x30  |
| `X(1)`   | `"Y"` / `"y"` | `true`     | 大小寫不敏感 |
| `X(1)`   | `"N"` / `"n"` | `false`    | 大小寫不敏感 |
| `A(1)`   | `"Y"` / `"y"` | `true`     | 大小寫不敏感 |
| `A(1)`   | `"N"` / `"n"` | `false`    | 大小寫不敏感 |

<br>

## Encode（bool → DISPLAY）

支援類型  

| PIC 類型  | bool 值  | 輸出 byte   | 顯示字元 |
| --------- | ------- | ------------ | ----- |
| `9(1)`    | `true`  | `0x31`       | `"1"` |
| `9(1)`    | `false` | `0x30`       | `"0"` |
| `X(1)`    | `true`  | `0x59`       | `"Y"` |
| `X(1)`    | `false` | `0x4E`       | `"N"` |
| `A(1)`    | `true`  | `0x59`       | `"Y"` |
| `A(1)`    | `false` | `0x4E`       | `"N"` |
