# Level Number 88 : Condition Name
為某個欄位定義邏輯名稱，用於程式判斷。

## 使用情境
- 把特定值封裝成語意化條件，程式碼更可讀。
- 只會針對單一欄位。

## 範例
```cobol
77 EMP-STATUS       PIC X VALUE 'A'.
   88 ACTIVE           VALUE 'A'.
   88 INACTIVE         VALUE 'I'.

IF ACTIVE
   DISPLAY "Employee is active"
END-IF.

```

> `IF ACTIVE` 自動判斷 `EMP-STATUS = 'A'`，可讀性比直接比較更高。

<br>

88 層級本身不能帶有 `OCCURS` 子句，但它可以定義在一個帶有 `OCCURS`（或屬於其下屬）的變數之後。

```cobol
01 ORDER-TABLE.
   05 ORDER-STATUS PIC X OCCURS 5 TIMES.
      88 ORDER-SHIPPED   VALUE 'S'.
      88 ORDER-PENDING   VALUE 'P'.
      88 ORDER-CANCELLED VALUE 'C'.
```
