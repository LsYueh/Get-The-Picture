# Level Number 77 : Standalone Variable (單一變數)
定義單獨的變數，不屬於任何群組。

<br>

## 使用情境
- 常用於計數器、旗標（flag）、暫存變數。
- 不需要群組，直接存取。

## 範例
```cobol
77 TOTAL-EMP        PIC 9(5) VALUE 0.
77 IS-MANAGER       PIC X VALUE 'N'.

```

> `TOTAL-EMP` 和 `IS-MANAGER` 是完全獨立的變數，可直接使用。

<br>

## 支援說明

> ⚠️ 目前不支援在 Copbook 內有 `77層級` 宣告的 Elementary Data Item 。
